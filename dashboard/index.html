C'est quoi la différence entre un incident et un accident du travail selon la loi marocaine ?
AI
Bonne question ! Selon le Code du Travail marocain (Dahir 2003) :

Incident : événement indésirable sans blessure grave — exemple : chute d'outil sans blessé.
Accident du travail : événement soudain causant une lésion corporelle pendant le travail — déclaration CNSS obligatoire dans les 48h.

🧠 Exemple concret à Tanger : dans une usine automobile, une fuite d'huile sur le sol = incident. Si un technicien glisse et se blesse = accident du travail, déclaration CNSS obligatoire.
